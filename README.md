# ウミウシAutoChess

ウミウシの駒を集めて配置し、全5ステージのクリアを目指す一人用オートチェスです。  
ガチャで候補を引き、駒を買い、同じ駒を盤面に3体並べて進化させながら、敵の属性や配置に合わせて部隊を組み替えて戦います。

## 遊ぶ

- ゲーム本編: https://koide-takeshi.github.io/make_autochess/
- ルール説明: https://koide-takeshi.github.io/make_autochess/rules.html
- ローカルで遊ぶ場合: `index.html` をブラウザで開いてください。

## 内容

- 一人用のステージ攻略型オートチェス
- 全5ステージ
- ガチャによる駒の購入
- 控え欄と盤面を使った部隊編成
- 同じ駒3体によるレベルアップ
- 炎、森、土、水の属性
- 戦闘中のダメージ集計
- ゲームクリア時の演出画像

## 基本ルール

1. 「駒を買う」から駒を購入します。
2. 控え欄や部隊欄の駒を選び、盤面に配置します。
3. 「戦闘開始」を押すと自動で戦闘します。
4. 勝っても負けてもゴールドを獲得します。
5. ガチャ、売却、配置変更、レベルアップ、属性相性を使って次のステージに備えます。
6. 5ステージ目を突破するとゲームクリアです。

詳しいルールは [ルール説明](https://koide-takeshi.github.io/make_autochess/rules.html) にまとめています。

## 操作

- 駒をクリック: 選択
- 選択中に別のマスをクリック: 配置または入れ替え
- 同じ駒をもう一度クリック: 選択解除
- 部隊欄の売値ボタン、または控え欄横の売却ボタン: その駒を売却
- Give up: 最初からやり直し
