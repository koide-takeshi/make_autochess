# 制作期間

このプロジェクトは、空き時間にcodexを使って少しずつ制作しました。

作業量としては仕事に換算すると約7日程度です。

主な作業内容は以下です。

* 一人用オートチェスとしての基本ルール設計
* ステージ進行、ガチャ、購入、売却、控え欄、配置操作の実装
* 自動戦闘、攻撃モーション、ダメージ集計、勝敗処理の実装
* 同じ駒3体によるレベルアップ要素の追加
* 炎、森、土、水の属性相性と敵属性の調整
* 敵の強さ、獲得ゴールド、ガチャ回数などのバランス調整
* ウミウシの味方駒、海の敵、海底神殿フィールドなどの画像素材作成
* ゲームクリア演出とご褒美画像表示の追加
* ルール説明ページと README の整理
